# 🛡️ Анти ИИ Щит

**Бесплатная офлайн-защита от телефонных мошенников**

## 📱 О проекте

«Анти ИИ Щит» — это социальный проект, который помогает распознать мошенников и защитить себя от обмана. Приложение работает полностью офлайн, не собирает данные и бесплатно навсегда.

## 🎯 Возможности

| Функция | Описание |
|---------|----------|
| 🗣️ **АВТООТВЕТЧИК** | 9 фраз для безопасного разговора с мошенником |
| 🧠 **ИИ: АНАЛИЗ УГРОЗЫ** | Проверка текста сообщения на признаки мошенничества |
| ⚠️ **SOS-РЕЖИМ** | Красный экран с видео-инструкцией «Положите трубку» |
| 🎓 **ТРЕНАЖЁР** | 11 уроков, чтобы научиться распознавать обман |
| 📢 **ОПАСНЫЕ ФРАЗЫ** | 20 фраз мошенников с озвучкой |
| 🔍 **ПРОВЕРКА ЗВОНКА** | Чек-лист из 8 признаков мошенничества |

## 🔧 Как использовать

1. **АВТООТВЕТЧИК** — нажмите во время подозрительного звонка
2. **ИИ: АНАЛИЗ УГРОЗЫ** — вставьте текст сообщения, ИИ определит мошенника
3. **SOS** — красный экран с инструкцией, кнопка СТОП с таймером
4. **УПРАЖНЕНИЯ** — 11 уроков, 20 опасных фраз, чек-лист
